适用于文件同步和新增文件(自动创建目录)，同步多个项目, 支持自动提交svn

1.需要安装chrome谷歌浏览器，因为html依赖于chrome


2.配置文件为conf.ini
2.1 在配置文件中[files]下添加对应的项目目录
# 项目名 = 项目目录路径, 例子： "测试1" = "D:\test1"

2.2 在配置文件中[ext_open]下可添加允许同步的后缀名
# 放开可同步的后缀名文件，多个用|隔开, ".xxx|.xxx|.xxx"


3.使用方法：
 启动exe，会显示gui界面，自动加载项目列表
 1.左侧栏上方select下拉框，选择对应的源项目
 2.左侧栏中间的svn自动提交框，确定是全量复制，可勾选并填写svn提交备注，同步后会自动提交svn
 3.左侧栏下方input框，输入所需同步的文件名，文件跟文件之间需换行，如下：
   test.php
   test1.php
   test.sql
 4.勾选右侧栏所需要同步的目标项目
 5.点击开始同步按钮即可
 6.完成后右上角关闭即可


last:程序占用的端口号是8080，有go环境可以自行修改，重新编译即可
